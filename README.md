# 🚀 LastMinute AI

> **Your AI Productivity Companion**

LastMinute AI is an AI-powered productivity assistant that helps users prioritize tasks, manage deadlines, generate intelligent action plans, receive AI-powered productivity advice, track goals, and visualize upcoming work through an interactive calendar.

## 📌 Problem Statement

Build an AI-powered productivity assistant that helps users organize work, prioritize tasks intelligently, improve productivity, and manage deadlines using Google's AI technologies.

---

## ✨ Features

### 🤖 AI Task Intelligence

* AI-based task prioritization
* Estimated completion time
* AI-generated task breakdown
* Personalized productivity advice
* AI reminders

### 📅 Smart Planning

* Smart schedule generation
* Calendar integration
* Deadline tracking
* Due today / Due tomorrow alerts
* Overdue task detection

### 📊 Productivity Dashboard

* Pending task summary
* Remaining work estimation
* AI productivity insights
* Focus task recommendation

### 🎯 Goal Tracking

* Create goals
* Track progress
* Increment progress
* Goal completion

### 🔔 Notifications

* Browser notifications
* Task added notifications
* Due today reminders
* Due tomorrow reminders

### ✅ Task Management

* Create tasks
* Mark Complete / Undo
* Delete tasks
* Store tasks in MongoDB

## 🚀 Installation

### Clone Repository

```bash
git clone <your-github-repository-link>
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the backend folder.

```env
MONGO_URI=YOUR_MONGODB_URI

GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```
